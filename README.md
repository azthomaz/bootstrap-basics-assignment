# bootstrap-basics-assignment
This is my first React-Bootstrap project. It includes a responsive layout of a Registration Form for a MusicHub, mock-up data &amp; a Navigation Bar.

## Features
- Registration form with Bootstrap styling
- Responsive navigation bar
- Music-themed hero images and sample submission table
- Mobile-friendly layout

## Installation
```bash
npm install
```

## Run the project
```bash
npm run dev
```

Then open the local URL shown in the terminal in your browser.
